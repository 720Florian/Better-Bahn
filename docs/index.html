Better Bahn

Finde günstigere Split-Ticket-Optionen für Deutsche Bahn Verbindungen

Download APK

Funktionen

Split-Ticket Analyse

Findet automatisch die günstigste Kombination von Einzeltickets für deine Strecke.

BahnCard Unterstützung

Berücksichtigt deine BahnCard 25/50 für 1. und 2. Klasse bei der Preisberechnung.

Deutschland-Ticket

Integration für Deutschland-Ticket Inhaber - Nahverkehrsstrecken werden automatisch berücksichtigt.

Direkte Buchung

Direkte Links zu bahn.de für jedes Ticket - einfach und schnell buchen.

100% Lokal

Open Source
